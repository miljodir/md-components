## Usage Examples

### Components
Use radius tokens for consistent component styling:
```css
.card {
  border-radius: var(--md-radius-md);
}

.button {
  border-radius: var(--md-radius-sm);
}

.pill {
  border-radius: var(--md-radius-xl);
}
```

### Guidelines
- **XS (2px)**: Small interactive elements, tags
- **SM (4px)**: Buttons, inputs, small cards
- **MD (8px)**: Cards, modals, containers
- **LG (16px)**: Large containers, sections
- **XL (360px)**: Pills, badges, fully rounded elements
