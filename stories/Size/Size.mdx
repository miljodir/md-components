## Usage Examples

### Spacing
Use size tokens for consistent spacing between elements:
```css
.element {
  margin: var(--md-size-16);
  padding: var(--md-size-8) var(--md-size-16);
  gap: var(--md-size-12);
}
```

### Component Sizing
Use size tokens for component dimensions:
```css
.icon {
  width: var(--md-size-24);
  height: var(--md-size-24);
}

.button {
  padding: var(--md-size-12) var(--md-size-24);
}
```
