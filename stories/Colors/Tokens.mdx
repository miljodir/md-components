# Tokens

We use semantic design tokens in css for most of our css variables. These tokens help maintain a consistent design language across our components and make it easier to update styles in the future.<br/>
We try to give these tokens <strong>meaningful names</strong> that reflect their purpose and usage within the design system. <br/><br/>
These tokens use variables defined in the [Colors section](/docs/introduction-colors-tokens-colors--docs)
